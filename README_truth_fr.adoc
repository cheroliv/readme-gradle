Plugin Gradle publié sous licence Apache 2.0 sur le link:https://plugins.gradle.org[Gradle Plugin Portal].

Il génère automatiquement des fichiers `README.adoc` compatibles GitHub depuis des fichiers source `README_truth.adoc` contenant des blocs PlantUML natifs.

== 1. Problème résolu

GitHub ne rend pas nativement la syntaxe PlantUML dans les fichiers AsciiDoc.
Un bloc `[plantuml]` affiché tel quel sur github.com n'est pas rendu en image.

Ce plugin automatise la transformation via GitHub Actions :

[plantuml, problem-solution, png]
----
@startuml
skinparam componentStyle rectangle

package "Ce que tu écris" {
  [README_truth.adoc\n(blocs plantuml natifs)] as src #LightGreen
}

package "Ce que GitHub affiche" {
  [README.adoc\n(image::...png références)] as out #LightBlue
}

package "Ce que le visiteur voit" {
  [Diagrammes rendus\ndans le navigateur] as web #LightYellow
}

src --> out : plugin génère\n(GitHub Actions)
out --> web : GitHub rend\nnativement
@enduml
----

== 2. Philosophie : source de vérité unique

`README_truth.adoc` est le **seul fichier que tu édites**.
`README.adoc` est un artefact généré — jamais édité à la main.

[plantuml, source-of-truth, png]
----
@startuml
skinparam defaultTextAlignment center

rectangle "Tu édites" #LightGreen {
  file "README_truth.adoc" as src
  file "README_plantuml_fr.adoc" as srcFr
}

rectangle "CI génère" #LightBlue {
  file "README.adoc" as out
  file "README_fr.adoc" as outFr
  folder ".github/workflows/readmes/images/en/" as imgEn
  folder ".github/workflows/readmes/images/fr/" as imgFr
}

src --> out
src --> imgEn
srcFr --> outFr
srcFr --> imgFr
@enduml
----

== 3. Fonctionnement

=== Vue d'ensemble

[plantuml, overview-sequence, png]
----
@startuml
actor Developer
participant "GitHub" as GH
participant "GitHub Actions" as CI
participant "processReadme\n(Gradle Task)" as proc
participant "commitGeneratedReadme\n(Gradle Task)" as commit
participant "JGit" as jgit

Developer -> GH : git push README_plantuml*.adoc
GH -> CI : déclenche workflow
CI -> proc : ./gradlew commitGeneratedReadme
proc -> proc : détecte blocs [plantuml,name,...]
proc -> proc : génère .github/workflows/readmes/\nimages/{lang}/name.png
proc -> proc : réécrit README.adoc\navec image::...png[]
proc -> commit : dependsOn résolu
commit -> jgit : git add + commit + push
jgit -> GH : README.adoc + images commités
GH --> Developer : README visible ✓
@enduml
----

=== Diagramme d'activité

[plantuml, activity, png]
----
@startuml
start

:Lecture de readme-truth.yml\n(injecté depuis le secret CI);

:Scan des README_plantuml*.adoc\ndans sourceDir;

if (Fichiers trouvés ?) then (non)
  :Warning — aucun fichier source;
  stop
else (oui)
  :Détection de la langue\ndepuis le suffixe _xx;
endif

repeat
  :Extraction des blocs\n[plantuml, name, ...];

  if (Blocs trouvés ?) then (oui)
    repeat
      :Génération PNG\nbuild/img/{lang}/name.png;
      :Copie vers\n.github/workflows/readmes/images/{lang}/;
      :Remplacement du bloc par\nimage::.github/.../name.png[name];
    repeat while (autres blocs ?)
  endif

  :Écriture README{_lang}.adoc;
repeat while (autres fichiers source ?)

:git add README*.adoc\n+ .github/workflows/readmes/images/;
:git commit + push\nvia JGit (sans git système);

stop
@enduml
----

== 6. Configuration

La configuration est externalisée dans `readme-truth.yml`.
Ce fichier ne contient **jamais** le token en clair dans le repo.
Son contenu complet (token inclus) est stocké dans le secret GitHub `README_GRADLE_PLUGIN`.

=== Template `readme-truth.yml`

[source,yaml]
----
source:
  dir: "."                    # <1>
  defaultLang: "en"           # <2>

output:
  imgDir: ".github/workflows/readmes/images"  # <3>

git:
  userName: "github-actions[bot]"
  userEmail: "github-actions[bot]@users.noreply.github.com"
  commitMessage: "chore: generate readme [skip ci]"
  token: "<YOUR_GITHUB_PAT>"  # <4>
  watchedBranches:
    - "main"
    - "master"
----
<1> Répertoire contenant les `README_plantuml*.adoc`
<2> Langue par défaut quand aucun suffixe `_xx` n'est présent
<3> Dossier des images générées — versionné git
<4> Remplacé par le vrai PAT uniquement dans le secret GitHub

=== Injection du secret en CI

[source,yaml]
----
- name: Inject plugin config
  run: echo "${{ secrets.README_GRADLE_PLUGIN }}" > readme-truth.yml

- name: Generate README and commit via JGit
  run: ./gradlew -q -s commitGeneratedReadme --no-daemon
----

== 8. Workflow GitHub Actions complet

[source,yaml]
----
name: Generate README from PlantUML sources

on:
  push:
    branches:
      - main
      - master
    paths:
      - "README_plantuml*.adoc"

  workflow_dispatch:

jobs:
  generate-readme:
    runs-on: ubuntu-latest
    permissions:
      contents: write

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'
          cache: gradle

      - name: Grant execute permission for gradlew
        run: chmod +x gradlew

      - name: Inject plugin config
        run: echo "${{ secrets.README_GRADLE_PLUGIN }}" > readme-truth.yml

      - name: Generate README and commit via JGit
        run: ./gradlew -q -s commitGeneratedReadme --no-daemon
----
