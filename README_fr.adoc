Plugin Gradle publié sous licence Apache 2.0 sur le link:https://plugins.gradle.org[Gradle Plugin Portal].

Il génère automatiquement des fichiers `README.adoc` compatibles GitHub depuis des fichiers source `README_plantuml.adoc` contenant des blocs PlantUML natifs.

== 1. Problème résolu

GitHub ne rend pas nativement la syntaxe PlantUML dans les fichiers AsciiDoc.
Un bloc `[plantuml]` affiché tel quel sur github.com n'est pas rendu en image.

Ce plugin automatise la transformation via GitHub Actions :

image::.github/workflows/readmes/images/fr/problem-solution.png[problem-solution]

== 2. Philosophie : source de vérité unique

`README_plantuml.adoc` est le **seul fichier que tu édites**.
`README.adoc` est un artefact généré — jamais édité à la main.

== 6. Configuration

La configuration est externalisée dans `readme-plantuml.yml`.
Ce fichier ne contient **jamais** le token en clair dans le repo.
Son contenu complet (token inclus) est stocké dans le secret GitHub `README_GRADLE_PLUGIN`.

=== Template `readme-plantuml.yml`

[source,yaml]
----
source:
  dir: "."                    # <1>
  defaultLang: "en"           # <2>

output:
  imgDir: ".github/workflows/readmes/images"  # <3>

git:
  userName: "github-actions[bot]"
  userEmail: "github-actions[bot]@users.noreply.github.com"
  commitMessage: "chore: generate readme [skip ci]"
  token: "<YOUR_GITHUB_PAT>"  # <4>
  watchedBranches:
    - "main"
    - "master"
----
<1> Répertoire contenant les `README_plantuml*.adoc`
<2> Langue par défaut quand aucun suffixe `_xx` n'est présent
<3> Dossier des images générées — versionné git
<4> Remplacé par le vrai PAT uniquement dans le secret GitHub

=== Injection du secret en CI

[source,yaml]
----
- name: Inject plugin config
  run: echo "${{ secrets.README_GRADLE_PLUGIN }}" > readme-plantuml.yml

- name: Generate README and commit via JGit
  run: ./gradlew -q -s commitGeneratedReadme --no-daemon
----

== 8. Workflow GitHub Actions complet

[source,yaml]
----
name: Generate README from PlantUML sources

on:
  push:
    branches:
      - main
      - master
    paths:
      - "README_plantuml*.adoc"

  workflow_dispatch:

jobs:
  generate-readme:
    runs-on: ubuntu-latest
    permissions:
      contents: write

    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'
          cache: gradle

      - name: Grant execute permission for gradlew
        run: chmod +x gradlew

      - name: Inject plugin config
        run: echo "${{ secrets.README_GRADLE_PLUGIN }}" > readme-plantuml.yml

      - name: Generate README and commit via JGit
        run: ./gradlew -q -s commitGeneratedReadme --no-daemon
----
